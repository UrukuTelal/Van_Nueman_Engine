## Slide 7 — The Cognitive Loop

1. **Perception** — Agent samples voxel environment.
2. **Context Building** — Pillar AI generates governance‑aware context.
3. **LLM Reasoning** — Local model produces decisions.
4. **Pillar Update** — Agent's internal state vector changes.
5. **Action** — Agent acts in voxel world.
6. **World Update** — Environment evolves.

This forms a closed cognitive‑simulation loop.
