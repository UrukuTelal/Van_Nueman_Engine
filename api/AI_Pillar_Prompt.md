[cite_start]Governance is not achieved through traditional voting but through **Constraint-Based Decision Validation**[cite: 23].

* **Formal Decision Rule:** A proposed change ($\Delta$) is valid **if and only if** the resulting state for every pillar ($p$) remains within its defined acceptable range:

* [cite_start]**Conflict Resolution:** When units conflict, the system chooses the outcome that **minimizes total system harm** ($\sum H_{sys}$) while **preserving total system weight** ($\sum W_{total}$)[cite: 25, 83].
